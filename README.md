# Legacy Limited - Landing Page

## Overview
Ultra-premium, minimalist landing page for Legacy Limited (BVI) - a luxury concierge and hospitality company.

## Brand Guidelines Implemented
- **Color Palette**: Strict black (#000000) and white (#FFFFFF) only
- **Typography**: Glacial Indifference font family
- **Design Philosophy**: Minimal, spacious, editorial, high contrast
- **Tone**: Calm, assured, elegant, exclusive without arrogance

## Features
- Fully responsive design
- Smooth scroll animations
- Subtle parallax effects
- Clean, minimalist UI
- SEO-optimized structure
- Fast loading performance

## How to View
1. Simply open `index.html` in any modern web browser
2. Or use a local server:
   - VS Code: Use Live Server extension
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve`

✓ Large typography with generous spacing
